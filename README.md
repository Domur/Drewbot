Java Discord bot that provides several useful and fun features, such as a leaderboard to keep track of the members who send the most messages.

List of commands: </br>
$help:  Lists these commands and explanations in the chat. </br>
$userinfo:  Displays user's account creation and server join date. </br>
$leaderboard:  Displays a leaderboard of chat statistics. </br>
$mashup:  Generates a random music mashup. </br>
$join:  Joins the voice channel. </br>
$leave:  Leaves the voice channel. </br>
$poll: Creates a poll with the parameters specified of, question, optional answers, and time limit. </br>
$roll: Rolls a 20 sided die. Can use $roll [number] for specific amount </br>
